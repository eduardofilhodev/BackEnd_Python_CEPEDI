# Sistema de Prontuário Hospitalar

Projeto simples em **Python** para gerenciamento de pacientes em um hospital.  

## Conceitos Utilizado
- Classes e objetos
- Herança
- Polimorfismo
- Encapsulamento
- Composição
- Associação
- Exceção personalizada
- Validação de dados

## Funcionalidades

O sistema possui um menu no terminal que permite:

1. Cadastrar paciente  
2. Exibir informações de um paciente  
3. Exibir informações do médico  
4. Exibir informações do enfermeiro  
5. Sair do sistema
